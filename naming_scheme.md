# Functions
camelCase
```cpp
thisIsAFunction
```

# Classes
PascalCase
```cpp
ThisIsAClass
```

## Class Variables
snake_case with leading underscore
```cpp
this_is_a_class_var_
```

# Non-Class Variables
snake_case
```cpp
this_is_a_var
```

# Const Variables
camelCase, but start with a k
```
const kThisIsAConstVariable
```
# Constexpr & defines
TITLE_CASE
```cpp
THIS_IS_A_CONSTEXPR_OR_DEFINE
```

# Files
kebab-case
```
this-is-a-filename.cpp
```
